Email Input

input type="text"

This is an example application that illustrates how to utilize some features in HTML 5 to simplify email input for mobile users.

Take a look at this first example, it's a plain, ordinary text input field:

That's the way most web developers implement the email input field: without any special formatting instructions that tells the browser that an email address is being entered.

Sigh.

This approach is not a big deal for desktop browser users as the @ symbol used in ALL email addresses is easily accessible.

For Mobile users, on the other hand, the @ symbol is not usually exposed through the default soft keyboard found on most touch screen devices, so it takes extra work for mobile users to be able to type that character.

input type="email"

Because of mobile devices and touch screens, web developers should always (yes, I do mean allways) use the HTML 5 email input type as demonstrated by the following input field:

As you can see, when using this input type, the email entry keyboard displays on mobile devices, simplifying input.
